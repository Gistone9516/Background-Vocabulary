### 11c-3. 단가 낮은 구독에서 수수료 영향 — 3,500원 구독 1건당 계산식
- 유형: 근거·수치
- 출처: 비용수익_예측.md §5.3, "단가가 낮은 구독에서 수수료의 영향." 문단 및 코드블록(669~677행)
- 폼팩터: 재검토(가격·통화가 v2에서 바뀔 수 있으나 계산 방법론은 무관하게 유효)
- 원문:
> **단가가 낮은 구독에서 수수료의 영향.** 월 3,500원 구독 1건당 수수료를 계산하면:
>
> ```
> 수수료 = 3,500원 × 3.4% × (1 + VAT 0.10)
>        = 3,500 × 0.034 × 1.10
>        = 3,500 × 0.0374
>        = 130.9원  (약 131원)
> 실수령 = 3,500 - 131 = 3,369원  (결제액의 약 96.3%)
> ```

### 11c-4. VAT 포함 131원/건을 보수적 기준으로 채택 + 가상계좌 불리 판단
- 유형: 확정결정
- 출처: 비용수익_예측.md §5.3, 679행
- 폼팩터: 재검토
- 원문: > 출처(토스페이먼츠 PG 수수료)는 VAT 별도 3.4% 기준으로 "약 119원"을 예시하나(3,500 × 0.034 = 119원), VAT 10%를 포함하면 약 131원이다. 보고서 §4 마진 계산에는 **VAT 포함 131원/건(결제액의 약 3.74%)**을 보수적으로 쓴다. 가상계좌 결제는 건당 400원 정액이라 3,500원 구독에서는 카드 결제(3.74%)보다 불리하므로(400/3,500 ≈ 11.4%), **카드·간편결제 유도**가 마진에 유리하다.

### 11c-26. 6.2 R1 — 무료 풀 원가는 가격 아닌 소진율이 레버(BEP 민감도 수치)
- 유형: 근거·수치
- 출처: 비용수익_예측.md §6.2, 774~781행
- 폼팩터: 무관
- 원문: > §2의 규모별 비용표에서 저전환(1~2%) 구간 총원가의 압도적 비중은 **무료 사용자의 LLM 원가**다. §2 가정대로 무료 사용자가 주 7회 한도를 100% 소진하면, 무료 풀 원가는 사용자 수에 정비례해 깔리는 반면 매출은 전환된 소수에서만 나온다. 따라서:
>
> - **가격 레버(예: 3,000 → 3,500원, +17%)는 BEP를 거의 못 낮춘다.** §4 기준 BEP 전환율은 약 2.5% → 2.1%로 0.4%p만 이동. 이유는 분모(무료 풀 원가)가 가격과 무관하게 그대로이기 때문.
> - **소진율 레버는 BEP를 직접 끌어내린다.** 평균 소진율이 100% 가정에서 실측 50%로 내려가면 무료 LLM 원가가 절반이 되어 BEP 전환율이 약 1%대까지 하락(§4 민감도). **즉 "무료 한도 1회 조정"이 "가격 17% 인상"보다 손익에 크게 작용한다.**
>
> 근거(단가): DeepSeek V4 Flash 출력 $0.28/MTok, 입력(캐시 미스) $0.14/MTok, 캐시 히트 $0.0028/MTok(약 98% 할인) — DeepSeek API Docs(`https://api-docs.deepseek.com/quick_start/pricing`, asof 2026-06-27). 무료 풀은 flash 전용이므로 무료 원가는 이 단가에 세션당 토큰·세션 수를 곱한 값(§1·§2 계산식 참조).
