# 배경노트 (Vock note)

> **한 줄 정체:** AI에게 말 걸기 전, 그 분야의 말그릇(핵심 어휘)을 클릭만으로 쥐여주는 **크롬 확장(MV3)**. 이름은 한국어 '배경노트', 영어 'Vock note'(Voca + back + note). — 자기 전공 밖으로 무언가 만들려는 "교차전공 빌더(탑다운)"를 위한 어휘 준비 도구. 영어 웹 RAG로 근거를 잡아 한국어 말그릇으로 변환한다.

**현재 단계:** buildflow 구현 진행 중 — **태그→Keep · 로컬 라이브 통합 · 비용/티어 차등 · 다국어(ko·en·ja·zh) · 보안 하드닝 · 평가 §c/§d 채택 · 아키네이터 탐색 저장·이어서 진행(세션별 턴 예산, 주간 한도=탐색 시작 차감 — deepflow 3사이클 검증) · 난이도 화면 LLM 예시(`/preview`)·생성 리스트 복원 · **세션 정돈 UI(좌상단 버거 → 플로팅 선택 드로어[상단 모드·하단 최근세션] · 프로젝트(폴더)·스코프홈 · 세션 검색·고정·날짜버킷 · 전역 화면 전환 애니) + 담기→프라이머 카피 교정 · 프로젝트 맥락 전파(연결 턴·제목→아키네이터·정렬→상세, deepflow 검증)**까지 완료·검증**. 다음 = 프로덕션 배포(`docs/배경어휘-사이드탭-배포-가이드.md`). **기획·계획 문서는 전부 `docs/`로 이동됨.**

## 기획·설계 핵심

- **타깃·범위:** 교차전공 빌더(탑다운), 저위험(기술·창작·비즈니스). 고위험(의료·법률 개인판단)은 **부드러운 거부**(이유 설명 + 재시도 유도).
- **모델·RAG:** DeepSeek-V4(`flash` 무료 / `pro` 유료) + **RAG 전 티어 필수**. thinking OFF + term 단위 스트리밍. **영어 검색 → 한국어 출력**(검색 로케일은 진입 분류에서 1회 결정론적 확정).
- **요금제·비용 차등:** freemium(유료 pro 월 3,500원). 출력량·기능으로 차등(모델은 양 티어 flash 동일). 무료: **세션별** 좁히기 3턴·총 어휘 8개·상세 3회/세션·그룹별 +2 생성·**주 7회 탐색**(주간 한도는 탐색 시작=`/classify` 시점 차감, 1 탐색=주간 1회. 이어서 진행은 재과금 없음). 유료: 좁히기 8턴·총 어휘 32개·상세 무제한·"더 보기"·그룹별 +4 생성·**조건 재탐색**(검색창에 조건 입력하면 아키네이터 재진입, 기존 리스트에 추가)·"AI로 더 정리"(/summarize, 방향성 입력 + 상세 브리핑 출력)·**파일 첨부**(붙여넣은 문서=context_object) 해금(무료는 첨부 버튼이 잠금 표시, 드롭·클릭 시 하단 고지 토스트). 출처 RAG는 양 티어 동일. 전 LLM 호출 `max_tokens` 상한 + 전역 일일 캡 + **누적 어휘 캡(`maxTotal`)** 강제.
- **보안 하드닝(인증 없는 anti-abuse):** IP당 분/일 요청 레이트리밋(`RATE_PER_MIN`·`RATE_PER_DAY`, Upstash, x-tier·x-user-id 스푸핑 무관) · 입력 텍스트 길이 상한(`MAX_INPUT_CHARS`, 토큰 비용·인젝션 방어, 413) · 프롬프트 인젝션 가드(사용자 입력=데이터, 지시 무시) · CORS 확장 origin 잠금(`ALLOWED_EXT_ORIGIN`, 프로덕션). 전역 일일 캡과 함께 다층 방어. 진짜 엔타이틀먼트(서버 검증 paid)는 인증/결제 인프라 생길 때(Tier3).
- **운영 한도 = 전부 워커 env로 튜닝(코드 무수정):** 어휘 개수·`max_tokens`·주간 한도·전역 캡·좁히기 턴·상세 횟수·**누적 어휘 캡(`MAX_TOTAL_*`)·그룹 생성 수(`GROUP_GEN_*`)**가 `shared`의 `DEFAULT_LIMITS` 기본값 위에서 env로 덮어쓰기 가능(`.dev.vars`/wrangler `[vars]`). 클라이언트 관련 값(좁히기 턴·상세 횟수·주간 한도·누적 캡·그룹 생성 수)은 워커 `GET /config`로 확장이 런타임 fetch → **재빌드 없이 반영**.
- **★ 이식성 불변(1순위 제약):** Cloudflare Workers 지금 / AWS Lambda 나중을 **어댑터 추가만으로**. `core/`·`shared/`는 런타임 전역 0건(웹표준 + 인터페이스 DI), 런타임 특수성은 `adapters/`에만. `npm run guard`가 위반 시 빌드 실패.
- **진입 화면 UX:** 첫 방문 1회 튜토리얼 팝업(**4스텝 — 목적·이유 3스텝[Claude 채팅 모방·어휘카드 미니·"나" 중심 은하계 분야맵] + 사용방법**, 헤더 ? 재열람, localStorage 기억) · 예시 칩은 4언어 각 **~150개 광범위 저위험 도메인** 풀(`examples.ts`)에서 랜덤 5개 + "다른 예시" 새로고침 + 은은한 플로팅 · **진입 오로라**(하늘·청록 주연 보라 포인트, 3층 교차) · pro 전용 파일 첨부("+ 파일", 붙여넣은 문서=context_object).
- **권위 규칙:** 제품 동작·UX·필드 = `panel.html`(최신 기획). 인프라(로케일·RAG·안전 게이트) = 백엔드.

## 아키텍처 (`sidetab/`)

```
shared/    계약 SoT(타입·인터페이스·프롬프트). core는 이 인터페이스에만 의존
core/      런타임 무관 로직: llm(DeepSeek SSE를 StreamEvent로) · rag · locale · pipeline
adapters/  workers(Hono·SSE 직렬화·게이팅) · search-providers(Tavily 영어전용) · lambda(보류)
extension/ MV3 사이드패널(React) — cycle 2에서 panel.html 5화면 이식
```

## 실행 / 빌드 / 검증 (`sidetab/` 기준)

```bash
npm install
npm run check                                             # 타입체크 + 이식성 가드
node --env-file=.env --import tsx packages/scripts/e2e.ts # 실런타임 e2e(실 API 호출)
npm -w @sidetab/extension run build                       # 확장 빌드(dist/)
# 워커 로컬 실행: packages/adapters/workers 에서 wrangler dev (.dev.vars에 키)
```

## 현재 상태 / 다음

- **cycle 1 완료·검증:** 프론트↔백엔드 계약 싱크(SoT)를 shared/core/adapters에 반영. 아키네이터 종료 신호(enough/confidence) · 어휘 상세 3단+출처 · "더 보기" exclude 페이지네이션 · 고위험 거부 · 취소 signal · /summarize 유료 게이팅. 타입체크 + 이식성 가드 + 실런타임 e2e 전부 PASS.
- **cycle 2 완료·검증:** `panel.html` 5화면을 React 확장(`sidepanel/`)으로 이식(theme.css=panel.html verbatim) + 5엔드포인트 배선(DEV는 mock, 빌드는 실 워커). 헤드리스 렌더 검증 통과(entry→classify→narrow→recommend 스트리밍→terms→detail 출처→summary, 콘솔 에러 0). vite 빌드 PASS, dist에 아이콘 포함.
- **cycle 2.1/2.2 + 직접입력 + 복수선택 완료:** 어휘 선정 철학 전환(개론 용어에서 실무 전문 용어·함정으로) · 아키네이터 직접 입력 버튼 + 적응형 입력 UI · 아키네이터 선택지 **무제한 복수 선택**(2개 cap 제거, buildPrompt1/2에 통합 보기 억제 문구).
- **태그→Keep 전환 완료(C1·C2·C3):** 어휘 카드 태그 {알아/몰라/적용모름}을 **단일 Keep 토글**로 교체(C1). Summary 화면 제거 후 **담은 어휘(kept) 뷰**에 복사·공유·AI정리 흡수(C2). 진입 입력 텍스트를 키로 한 **이전 탐색 히스토리**를 `chrome.storage.local`에 저장하고 entry에서 재열람·복원(C3). 백엔드·계약 변경 0건(클라이언트 전용, 저장은 `sidepanel/history.ts`). 실브라우저(vite mock) end-to-end 검증: entry→narrow→terms(Keep)→kept→히스토리 복원, 앱 콘솔 에러 0.
- **로컬 라이브 통합 완료·검증:** 로컬 실 워커(`npm -w @sidetab/workers run dev`, `.dev.vars` 실 키)에 확장 dev(`VITE_WORKER_BASE=http://127.0.0.1:8787`)를 붙여 실 DeepSeek+Tavily로 end-to-end 검증. `/classify`(실 분류·선택지)·`/recommend`(실 어휘 스트리밍)·`/detail`(실 3단 개념·관련어) 동작, Keep·히스토리·상세캐시 영속까지 확인, 앱 콘솔 에러 0. `manifest.json` host_permissions에 로컬 워커 주소 추가됨.
- **비용 최적화/티어 차등 + 문장 줄바꿈 완료·검증:** 전 LLM 호출에 `max_tokens` 안전 상한(폭주 가드, free<paid) · 무료 4어휘/좁히기 3턴/상세 3회·세션/더보기·더깊이·요약 페이월 · 전역 일일 캡(`GLOBAL_DAILY_CAP`, recommend·detail·summarize) · anonymous 우회를 전역 캡으로 차단. 계약(SoT) 변경 2건: `LlmRequest.maxTokens`, `Pipeline.recommendStream/detail`에 `tier` 인자(`buildPrompt3` count). 출처 RAG는 양 티어 동일. 문장 끝 줄바꿈(`sentLines`)을 카드 one_line·why·좁히기 질문·hero 부제·거부·페이월 안내로 확대. **부드러운 거부(고위험 게이트)·페이월 톤 보존.** 실 API 무료 흐름 end-to-end 검증(4어휘·3턴·페이월 3종·출처 포함·줄바꿈), 검증 중 발견한 상세 `max_tokens` 과소(500 truncation) 버그 수정.

- **프로젝트 맥락 전파 = 어휘 자산의 능동 영향(연결 턴 + 제목→아키네이터 + 정렬→상세, deepflow 6렌즈 검증, 최신):** 프로젝트의 누적 kept 어휘와 제목이 이후 같은-프로젝트 세션을 능동적으로 바꾸게 했다(기획 해자 "어휘 자산의 영속"의 실체화). **deepflow 6렌즈+반박 패널**(낙관·프리모템·근거[웹]·메커니즘·오염·정체성)로 적대 검증 → silent 주입(누적 프라이밍·프로젝트 프라이머)은 느슨한 버킷(외부 세션 배정 가능, 주제 일관 보장 없음)에서 오염이라 기각, **사용자가 명시적으로 게이팅하는 형태**만 채택. 구현 3종: ① **연결 턴(`POST /relate` 신규, 좁히기 종료 직전 별도 단계)** — 같은 프로젝트·같은 도메인 kept가 `RELATE_MIN_KEPT`(3) 이상일 때만 LLM이 연결을 판정(`relevant`), 있으면 재인 질문 한 턴으로 드러내 사용자가 방향을 고르면 그 선택만 `user_condition`으로 추천에 실어(프라이밍=메커니즘2 안전판), 같은 프로젝트에서 담은 어휘는 `exclude`로 제외(dedup=메커니즘1). 가드 5: 조건부 삽입·"관련 없어요" 탈출·실패 degrade·같은 도메인 프리필터·kept 12캡. ② **제목 → 아키네이터 부드러운 사전** — 프로젝트 제목을 `classify`(첫 질문)+매 `nextBranch`에 약한 prior로 주입(막연한 입력을 가를 때만, 입력·선택이 명백히 다르면 그쪽 우선). ③ **정렬 시 → 상세 약한 반영** — 연결 턴에서 방향을 고른 경우(정렬 확정=`relateCond` 설정)에만 개념 설명(`detail`)에 제목+아는 어휘를 `connection_hint`로 약하게(이 어휘가 진짜 닿을 때만 whymine에 한 줄, 억지·지배 금지). 계약 가산만: `RelateIn/RelateOut`·`Prompt1In`/`Prompt2In`의 `project_context?`·`Prompt5In.connection_hint?`·`POST /relate` 라우트(주간 미집계 free), kept는 전부 `chrome.storage.local`(서버 스키마 무변경). i18n 4언어(`relate_*`). 정적(check+가드+확장 tsc 0+빌드) PASS, 5181 실 워커 검증. **비용(연결 판정 LLM 호출)은 핵심 기획이라 사용자가 감안하기로 확정.**
