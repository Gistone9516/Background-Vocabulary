# Background-Vocabulary — 배경노트 (Vock note)

AI 탑다운 학습 시대. 비전공자가 알아야 할 전문 배경어휘 지식을 뽑아주는 서비스

막연한 생각을 가진 사용자가 "설명(표현)하지 않고도" 클릭·재인만으로 의도를 결정화해, 자기 상황에 맞는 배경지식 어휘를 얻는 도구. 제품 핵심 축 = **표현(articulation) 비용 면제** 단일 축.

## v2 방향 (2026-07-20 결정)

- 폼팩터: 크롬 확장 → **웹사이트 + 크로스플랫폼 데스크톱 앱**. 데스크톱은 고유 기능 추가.
- 방법론: 소프트웨어 공학 명세 절차(요구사항 명세 → 아키텍처 설계 → 상세설계 → 구현 → 테스트). 디자인 패턴·계층 분리 기반의 유지보수 가능한 코드, 적층형 코딩 금지.
- v1 기획은 **요약 없이 원문 그대로 승계**(승계원장). 디자인 컨셉은 v1 유지(덜어내고 차분하게), 더 나은 안은 변경 후보로 검토.
- 결제·수익 모델은 사이드탭 제약 위 설계라 v2에서 재설계. free 티어 광고 BM 확정(2026-07-20), pro는 광고 없음.
- 백엔드 자산(Workers+Hono, D1 엔타이틀먼트, 서버 JWT, Google SSO, 프롬프트 파이프라인)은 폼팩터 무관이므로 재사용 우선 검토.
