# 배경어휘 — 확장 아이콘 세트

채택안: **어휘 카드 + 단일 AI 스파클** (단색 오커 배경)

## 파일
| 파일 | 크기 | 용도 |
|---|---|---|
| icon-16.png | 16×16 | 툴바·파비콘 |
| icon-32.png | 32×32 | 윈도우·고해상도 툴바 |
| icon-48.png | 48×48 | 확장 관리 페이지 |
| icon-128.png | 128×128 | **크롬 웹스토어 등록·설치** |
| icon-512.png | 512×512 | 마스터(스토어 프로모·여유분) |

## manifest.json
```json
"icons": {
  "16": "icons/icon-16.png",
  "32": "icons/icon-32.png",
  "48": "icons/icon-48.png",
  "128": "icons/icon-128.png"
},
"action": {
  "default_icon": {
    "16": "icons/icon-16.png",
    "32": "icons/icon-32.png"
  }
}
```

## 색 (정본 oklch · sRGB hex)
- 배경 메인 오커: `#B98935` — oklch(0.66 0.115 78)
- 잉크(글자·스파클): `#B0660C` — oklch(0.58 0.13 62)
- 형광펜 강조: `#E0A93E`
- 노트 크림: `#FBF2DB`

## 메모
- 스파클은 직접 그린 단순 대칭 4점 별(이모지 아님) — 저작권 무관. 특정 브랜드 마크와 형태·색이 달라 상표 이슈도 회피.
- 배경 그라데이션 제거(단색). 모든 PNG는 둥근 사각 + 투명 외곽.
